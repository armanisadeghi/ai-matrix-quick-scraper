<blockquote>
<p>Markdown parser done right.</p>
</blockquote>
<ul>
<li>Follows the <strong><a href="http://spec.commonmark.org/" rel="nofollow">CommonMark spec</a></strong> for baseline parsing</li>
<li>Configurable syntax: you can add new rules and even replace existing ones.</li>
<li>Pluggable: Adds syntax extensions to extend the parser (see the <a href="https://markdown-it-py.readthedocs.io/en/latest/plugins.html" rel="nofollow">plugin list</a>).</li>
<li>High speed (see our <a href="https://markdown-it-py.readthedocs.io/en/latest/other.html" rel="nofollow">benchmarking tests</a>)</li>
<li><a href="https://markdown-it-py.readthedocs.io/en/latest/other.html" rel="nofollow">Safe by default</a></li>
<li>Member of <a href="https://cloud.google.com/assured-open-source-software/docs/supported-packages" rel="nofollow">Google's Assured Open Source Software</a></li>
</ul>
<p>This is a Python port of <a href="https://github.com/markdown-it/markdown-it" rel="nofollow">markdown-it</a>, and some of its associated plugins.
For more details see: <a href="https://markdown-it-py.readthedocs.io" rel="nofollow">https://markdown-it-py.readthedocs.io</a>.</p>
<p>For details on <a href="https://github.com/markdown-it/markdown-it" rel="nofollow">markdown-it</a> itself, see:</p>
<ul>
<li>The <strong><a href="https://markdown-it.github.io" rel="nofollow">Live demo</a></strong></li>
<li><a href="https://github.com/markdown-it/markdown-it/blob/master/README.md" rel="nofollow">The markdown-it README</a></li>
</ul>

<h2>Usage</h2>
<h3>Python API Usage</h3>
<p>Render markdown to HTML with markdown-it-py and a custom configuration
with and without plugins and features:</p>
<pre lang="python3"><span class="kn">from</span> <span class="nn">markdown_it</span> <span class="kn">import</span> <span class="n">MarkdownIt</span>
<span class="kn">from</span> <span class="nn">mdit_py_plugins.front_matter</span> <span class="kn">import</span> <span class="n">front_matter_plugin</span>
<span class="kn">from</span> <span class="nn">mdit_py_plugins.footnote</span> <span class="kn">import</span> <span class="n">footnote_plugin</span>

<span class="n">md</span> <span class="o">=</span> <span class="p">(</span>
    <span class="n">MarkdownIt</span><span class="p">(</span><span class="s1">'commonmark'</span> <span class="p">,{</span><span class="s1">'breaks'</span><span class="p">:</span><span class="kc">True</span><span class="p">,</span><span class="s1">'html'</span><span class="p">:</span><span class="kc">True</span><span class="p">})</span>
    <span class="o">.</span><span class="n">use</span><span class="p">(</span><span class="n">front_matter_plugin</span><span class="p">)</span>
    <span class="o">.</span><span class="n">use</span><span class="p">(</span><span class="n">footnote_plugin</span><span class="p">)</span>
    <span class="o">.</span><span class="n">enable</span><span class="p">(</span><span class="s1">'table'</span><span class="p">)</span>
<span class="p">)</span>
<span class="n">text</span> <span class="o">=</span> <span class="p">(</span><span class="s2">"""</span>
<span class="s2">---</span>
<span class="s2">a: 1</span>
<span class="s2">---</span>

<span class="s2">a | b</span>
<span class="s2">- | -</span>
<span class="s2">1 | 2</span>

<span class="s2">A footnote [^1]</span>

<span class="s2">[^1]: some details</span>
<span class="s2">"""</span><span class="p">)</span>
<span class="n">tokens</span> <span class="o">=</span> <span class="n">md</span><span class="o">.</span><span class="n">parse</span><span class="p">(</span><span class="n">text</span><span class="p">)</span>
<span class="n">html_text</span> <span class="o">=</span> <span class="n">md</span><span class="o">.</span><span class="n">render</span><span class="p">(</span><span class="n">text</span><span class="p">)</span>

<span class="c1">## To export the html to a file, uncomment the lines below:</span>
<span class="c1"># from pathlib import Path</span>
<span class="c1"># Path("output.html").write_text(html_text)</span>
</pre>
<h3>Command-line Usage</h3>
<p>Render markdown to HTML with markdown-it-py from the
command-line:</p>
<pre lang="console"><span class="go">usage: markdown-it [-h] [-v] [filenames [filenames ...]]</span>

<span class="go">Parse one or more markdown files, convert each to HTML, and print to stdout</span>

<span class="go">positional arguments:</span>
<span class="go">  filenames      specify an optional list of files to convert</span>

<span class="go">optional arguments:</span>
<span class="go">  -h, --help     show this help message and exit</span>
<span class="go">  -v, --version  show program's version number and exit</span>

<span class="go">Interactive:</span>

<span class="gp">  $ </span>markdown-it
<span class="go">  markdown-it-py [version 0.0.0] (interactive)</span>
<span class="go">  Type Ctrl-D to complete input, or Ctrl-C to exit.</span>
<span class="go">  &gt;&gt;&gt; # Example</span>
<span class="go">  ... &gt; markdown *input*</span>
<span class="go">  ...</span>
<span class="go">  &lt;h1&gt;Example&lt;/h1&gt;</span>
<span class="go">  &lt;blockquote&gt;</span>
<span class="go">  &lt;p&gt;markdown &lt;em&gt;input&lt;/em&gt;&lt;/p&gt;</span>
<span class="go">  &lt;/blockquote&gt;</span>

<span class="go">Batch:</span>

<span class="gp">  $ </span>markdown-it<span class="w"> </span>README.md<span class="w"> </span>README.footer.md<span class="w"> </span>&gt;<span class="w"> </span>index.html
</pre>
